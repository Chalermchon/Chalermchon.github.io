CSS specificity


คือ การที่นำ CSS selector แต่ละชนิดมาใช้
โดยที่แต่ละตัวก็มีความสำคัญต่างกัน

CSS selector

  • Element Selector

    • คือ Tag ต่างๆ ซึ่งจะทำให้ทุก Tag มี Style เป็นไปตามนั้น

    มีความสำคัญต่ำสุด

    EX: p { ... }

  • Class Selector

    • คือ การระบุที่ Attribute Name "Class" ของ Tag นั้นๆ ตามชื่อนั้นๆ

    มีความสำคัญขึ้นมามากกว่า Element

    EX: .sth-class { ... }

  • ID Selector

    • คือ การระบุที่ Attribute Name "ID" ของ Tag นั้นๆ ตามชื่อนั้นๆ

    มีความสำคัญขึ้นมามากกว่า Class

    EX: #sth-id { ... }
